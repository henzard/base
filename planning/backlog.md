# Backlog

> Prioritized list of work. Move items to `tasks.*` when breaking down a feature. Ref: [GSD roadmap](https://github.com/gsd-build/get-shit-done), [BMAD epics/stories](https://github.com/bmad-code-org/BMAD-METHOD).

## Format

- **ID**: Short id (e.g. B1, B2).
- **Title**: One line.
- **Priority**: P0 (must) | P1 (should) | P2 (could).
- **Status**: Backlog | Ready | In progress | Done.

## Items

| ID  | Title | Priority | Status   |
|-----|--------|-----------|----------|
| B1  | _Example: Add user login_ | P0 | Backlog |
| B2  | _Example: Add password reset_ | P1 | Backlog |
